House Price Prediction System

Overview

This project uses Machine Learning models and ensemble voting techniques to analyze the California Housing dataset and generate predictive outputs based on housing-related features.

Features

- Dataset preprocessing and splitting
- Base model creation and training
- Individual model accuracy evaluation
- Ensemble learning using Hard Voting and Soft Voting
- Performance comparison of multiple models

Algorithms Used

- Logistic Regression
- Decision Tree
- K-Nearest Neighbors (KNN)
- Voting Classifier

Dataset

- California Housing Dataset

Workflow

1. Load and preprocess the dataset
2. Split the dataset into training and testing sets
3. Train individual machine learning models
4. Calculate model accuracy
5. Apply Hard Voting and Soft Voting techniques
6. Compare prediction performance

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn

Project Structure

- "main.py" – Main implementation file
- Dataset files
- Output prediction results

Output

The project compares individual machine learning models with voting-based ensemble methods to evaluate prediction performance and accuracy.
