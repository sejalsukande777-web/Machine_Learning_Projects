Diabetes Detection System



Machine Learning project for predicting diabetes likelihood using classification algorithms and medical datasets.



Features



\- Data preprocessing and cleaning

\- Exploratory Data Analysis (EDA)

\- Feature scaling using StandardScaler

\- Multiple classification models:

&#x20; - Logistic Regression

&#x20; - K-Nearest Neighbors (KNN)

&#x20; - Decision Tree

\- Model evaluation using:

&#x20; - Accuracy

&#x20; - Precision

&#x20; - Recall

&#x20; - F1 Score

&#x20; - Confusion Matrix

\- Data visualization

\- Prediction system for new patient data



Tech Stack



\- Python

\- Pandas

\- NumPy

\- Scikit-learn

\- Matplotlib

\- Seaborn
