Customer Segmentation System

Machine Learning project for grouping customers into distinct segments using K-Means Clustering based on Annual Income and Spending Score.

Features

- Data preprocessing and cleaning
- Feature scaling using StandardScaler
- Elbow Method for optimal cluster selection
- K-Means Clustering implementation
- Customer segment analysis
- Cluster distribution visualization
- Export of clustered dataset to CSV

Tech Stack

- Python
- Pandas
- Scikit-learn
- Matplotlib

Project Structure

Customer Segmentation System/
│
├── main.py
├── Mall_Customers.csv
└── README.md

Run the Project

python main.py

Output

The project:

- Loads and preprocesses customer data
- Identifies customer segments using K-Means Clustering
- Generates an Elbow Method graph
- Displays cluster statistics and sample records
- Creates "Mall_Customers_with_clusters.csv" containing cluster labels
